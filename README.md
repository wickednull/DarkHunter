## About The Project


DarkHunter is a powerful, extensible, and user-friendly web vulnerability scanner designed to help security professionals and developers identify and mitigate security risks in their web applications.

### Built With

*   [Python](https://www.python.org/)
*   [customtkinter](https://github.com/TomSchimansky/CustomTkinter)
*   [aiohttp](https://github.com/aio-libs/aiohttp)
*   [PyYAML](https://pyyaml.org/)
*   [colorama](https://github.com/tartley/colorama)

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

*   Python 3.7+
*   The dependencies listed in `requirements.txt`

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/wickednull/DarkHunter.git
    ```
2.  Install PIP packages
    ```sh
    pip install -r requirements.txt
    ```

## Usage

1.  Run the application
    ```sh
    python gui.py
    ```
2.  Configure your scan
    *   Add your target URL(s) in the "Target" tab.
    *   Define the scope of your scan in the "Scope" tab.
    *   Select the plugins you want to run in the "Plugins" tab.
    *   Configure advanced settings like concurrency and rate limiting.
3.  Start the scan
    *   Click the "Start Scan" button.
4.  View and export findings
    *   Live logs and findings will appear in the right-hand pane.
    *   Export your findings to various formats from the "Export" tab.
